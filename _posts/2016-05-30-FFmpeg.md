### 准备工作
在 Windows 构建无法脱离 MSys2，没有 MSys2 环境，无法生成几个重要的文件  
如何准备 MSys2 环境请参考 搭建 MSys2 环境  
需要使用 pacman 安装三个工具包  

    pacman -S diffutils
    pacman -S pkg-config
    pacman -S make

### 下载源码
使用下列命令下载 FFmpeg 的源代码  

    git clone https://git.ffmpeg.org/ffmpeg.git ffmpeg
    
### 配置编译
使用下列的命令进行配置  

    ./configure --enable-shared --prefix=./build
    
使用下面的命令进行编译和打包  

    make
    make install
